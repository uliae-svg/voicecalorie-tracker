# ◈ VoiceFood

> Голосовой ИИ-трекер питания с персональным диетологом на базе Llama 3.3 — скажите что съели, получите КБЖУ, микронутриенты и AI-анализ рациона.

## ✨ Особенности

- 🎙️ **Голосовой ввод на русском** — Web Speech API, никакого набора текста
- 🌐 **Автоматический перевод** — русский текст переводится в английский перед запросом к API
- ⚡ **КБЖУ в реальном времени** — калории, белки, жиры, углеводы появляются мгновенно
- 💊 **Микронутриенты** — витамины (A, C, D, E, K, B6, B12) и минералы (кальций, железо, магний, калий, натрий) с % дневной нормы
- 🏷️ **Метки здоровья** — цветные бейджи: Vegan, High Protein, Low Carb, Dairy Free и другие
- ⚠️ **Предупреждения** — высокое содержание сахара (> 15 г) или натрия (> 600 мг) выделяются красным
- 🧠 **AI-диетолог** — анализ дневника питания через Groq (Llama 3.3 70B) со стримингом в реальном времени
- 🥦 **Клетчатка** — отдельный счётчик с прогрессом (цель задаётся по профилю)
- 👤 **Профиль пользователя** — персональные цели КБЖУ по формуле Миффлина-Сан Жеора
- 📅 **Навигация по датам** — стрелки ‹ › для просмотра и заполнения прошлых дней
- 👆 **Свайп-жесты** — свайп влево удаляет запись, свайп вправо меняет приём пищи
- 📓 **Дневник питания** — записи за 30 дней сохраняются локально
- 📱 **Mobile-first** — адаптирован под любой экран

## 🚀 Как запустить

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/uliae-svg/voicecalorie-tracker.git
cd voicecalorie-tracker
```

### 2. Установите зависимости

```bash
npm install
```

### 3. Настройте переменные окружения

```bash
cp .env.example .env
```

Откройте `.env` и заполните ключи:

```env
# Edamam — https://developer.edamam.com/ → Nutrition Analysis
VITE_EDAMAM_APP_ID=ваш_app_id
VITE_EDAMAM_APP_KEY=ваш_app_key

# Groq — https://console.groq.com/ (бесплатно)
VITE_GROQ_API_KEY=ваш_groq_key
```

### 4. Запустите dev-сервер

```bash
npm run dev
```

Откройте [http://127.0.0.1:3000](http://127.0.0.1:3000) в **Chrome** или **Edge**.

> ⚠️ Web Speech API работает только в Chrome и Edge. Firefox не поддерживается.

## 🎙️ Как пользоваться

1. Откройте приложение в Chrome/Edge
2. Разрешите доступ к микрофону
3. Нажмите на аватар **👤** в верхнем левом углу и заполните профиль — приложение рассчитает персональные цели КБЖУ
4. Выберите приём пищи (Завтрак / Обед / Ужин / Перекус)
5. Нажмите кнопку **◉** и скажите что вы съели:
   - _«борщ 300 грамм»_
   - _«два яблока»_
   - _«гречка 200 грамм»_
6. Получите КБЖУ и метки здоровья — запись добавится в дневник
7. Нажмите **«Микронутриенты →»** для подробного разбора витаминов и минералов
8. Свайпните запись в дневнике влево чтобы удалить, вправо — чтобы сменить категорию
9. Нажмите **«🧠 Анализ от диетолога»** — получите персональный разбор рациона от AI
10. Используйте стрелки **‹ ›** рядом с датой чтобы посмотреть прошлые дни или заполнить пропущенный день

## 🔐 Безопасность

API-ключи хранятся в `.env` и **не попадают в Git**. Для продакшена рекомендуется проксировать запросы к Edamam через собственный бэкенд, чтобы ключи не были видны в браузере.
